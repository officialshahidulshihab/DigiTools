# DigiTools 🛠️

A modern digital tools e-commerce platform where users can browse, filter, and purchase premium digital products — all in one place.

🔗 Live Site: https://phdigitoolsbd.netlify.app/

---

## 🚀 Features

- 🛒 **Cart System** — Add, remove, and checkout products with a running total
- 🔔 **Toast Notifications** — Real-time feedback using React-Toastify for every cart action
- 🔄 **Product Filtering & Toggling** — Switch between Products and Cart views instantly

---

## 🧰 Technologies Used

- React.js
- Tailwind CSS
- DaisyUI
- JavaScript (ES6+)
- React-Toastify
